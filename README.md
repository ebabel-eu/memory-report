# Memory report

Use this Python script to view current Mac OS memory usage in MB.

## Sample usage

```
/usr/bin/python ./memory-report.py
```

## Sample output

```
Wired Memory:		1345 MB
Active Memory:		919 MB
Inactive Memory:	906 MB
Free Memory:		16 MB
Real Mem Total (ps):	2698.527 MB
```
